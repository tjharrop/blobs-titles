Why we moved to figma

  • We'd heard good things
  • We didn't like being mac only
  • Sketch had a steep learning curve
  • People had to combine sketch with other things
  • We really had heard good things

Good 👍

  • Multiple concurrent users
  • Auto-layout
  • More interactive where sketch needed invision etc
  • Hover states & interactive components
  • Designers can create new components alone

Bad 👎

  • Autolayout limitations
  • Interactive component limitations
  • Sometimes there’s a lag
  • Designers can create new components alone

Meh 💔

  • It’s flat
  • The code isn’t usable code
  • Org accounts don’t work for gov’

Nice 👌

  • Sharing and distribution
  • Plugins & Community
  • Collab & Commenting
  • Non designers can use it
  • Platform agnostic / browser based
  • Lots of existing tutorials (official and unofficial)

What's next?

  • Adding more interactive components
  • Get more users using Figma (service / content / product managers)
  • Using Figma to communicate with DDS users
  • Move to design schematics, not renders
